# Main Entry

File: `src/main.jsx`

## Purpose

Bootstraps the React application into the DOM.

## What it does

- Imports the global stylesheet from `src/index.css`
- Imports the root `App` component
- Creates the React root on `#root`
- Renders the app inside `React.StrictMode`

## Notes

This is the first browser-executed React file after Vite loads the app.
