# Global Styles

File: `src/index.css`

## Purpose

Holds the app-wide styling entry point loaded by `src/main.jsx`.

## Role in the app

- imports the global CSS used by the whole application
- is the place where Tailwind and custom base styles are applied
- supports the dark glass-style UI used across login, dashboard, and profile screens

## Notes

The visible styling is mostly driven by Tailwind utility classes in the components and pages, with this file acting as the global stylesheet entry point.
