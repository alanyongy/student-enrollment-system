# Student Profile

File: `src/pages/StudentProfile.jsx`

## Purpose

Shows the currently signed-in student's personal and academic information.

## Data it fetches

- student profile from `/api/me/profile`

## UI sections

- profile header with initials and display name
- enrollment summary cards
- detailed profile fields
- academic information summary

## Dependencies

- `Navbar`
- `api`

## Notes

The same navigation bar is reused here and on the student dashboard so the student can move between profile and registration screens.
